Create a data frame

problem2_df = tibble(
  nrom = rnorm(10, mean = 0, sd = 1),
  logi_vector = nrom > 0,
  char_vector = c("a", "b", "c", "d", "e", "f", "g", "h", "i", "j"),
  fac_vector = factor(c("High", "Medium", "Low", "High", "High", "High", "Low", "Low", "Low", "Medium"))
)
view(problem2_df)

Calculate the mean of variables in the dataframe

Answer: mean function only works on nrom and logical vector but doesnt work on character vector and factor vector.

Apply the as.numeric function to the logical, character, and factor variables

convert_logi_vector = as.numeric(pull(problem2_df, var = logi_vector))
convert_logi_vector
convert_char_vector = as.numeric(pull(problem2_df, var = char_vector))
convert_char_vector
convert_fac_vector = as.numeric(pull(problem2_df, var = fac_vector))
convert_fac_vector

Answer: Applying the as.numeric function to the logical and factor variables is successful; however, it doesnt work successfully on character vector, showing that characters cannot be converted into numbers. And logical and factor variables can be converted into numbers and be used in other calculations.
